# 大模型数据处理系统 - 前端

## 技术栈

- Vue 3
- Vite
- Element Plus
- Vue Router

## 开发环境搭建

1. 安装依赖：
   ```bash
   npm install
   # 或者
   yarn install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   # 或者
   yarn dev
   ```

3. 构建生产版本：
   ```bash
   npm run build
   # 或者
   yarn build
   ```

## 项目结构

- `src/main.js` - 应用入口文件
- `src/App.vue` - 根组件
- `src/router` - 路由配置
- `src/views` - 页面组件
- `src/components` - 可复用组件
- `src/assets` - 静态资源
